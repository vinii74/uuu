Majors Overview

Computer Science (CS)

Focuses on programming, algorithms, and software development. Covers fundamental concepts of computation and information processing.

  • Software Engineering
  • Algorithms & Data Structures
  • Computer Architecture

Data Science (DS)

Emphasis on data analysis, machine learning, and statistical modeling. Teaches how to extract insights from complex data.

  • Machine Learning
  • Data Visualization
  • Big Data Technologies

Information Technology (IT)

Covers IT infrastructure, networks, and enterprise systems. Focuses on practical implementation of computing solutions.

  • Network Management
  • Web Technologies
  • System Administration
